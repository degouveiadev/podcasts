## ¿Cómo funciona?

### Con Node.JS:

Requiere Node.JS 12

- `npm install` para instalar las dependencias.
- `npm run dev` para el entorno de desarrollo.
- `npm run build && npm start` para el entorno de producción.

### Con Docker:

- `docker-compose up -d` para crear y ejecutar el contenedor.
